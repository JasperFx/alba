The main drag of the bustling metropolis of Alba, Missouri

Supercharging ASP.Net Core development

Use the Alba library in your testing projects to quickly write declarative specifications against the HTTP endpoint of an ASP.Net Core application by simulating an HTTP request in memory. Inspired by the "Scenario" testing mechanism in FubuMVC. Alba supports the CoreCLR runtime (Netstandard 1.6).
